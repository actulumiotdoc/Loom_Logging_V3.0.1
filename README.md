 ## Program Logging New version
**ฐานข้อมูล**
- เปลี่ยนจาก file system เป็น SQL Database *#มีการทำ Transaction เพื่อลดข้อผิดพลาดในการเขียนข้อมูล*

**รูปแบบข้อมูล**
- แยก table ข้อมูลระหว่าง Local Server กับ Cloud Server
  
**การตั้งค่าเริ่มต้น**
- เปลี่ยนจากไฟล์ตั้งค่า เป็นการ Query จากข้อมูลล่าสุด และนำข้อมูล timestamp มาเป็นตัวกำหนดค่าเริ่มต้น

**การส่งข้อมูล**
- Local Server ใช้ http node
- Cloud Server ใช้ Python Script file

**dashboard**
- เพิ่มหน้าตั้งค่าขนาดลูกกลิ้งและจำนวนเฟืองเกีบร์
- แก้ไขหน้า logging อุปกรณ์

**logging อุปกรณ์**
- เก็บรวมใน SQL ทั้งหมด
